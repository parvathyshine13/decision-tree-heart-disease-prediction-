# 🌳 Decision Tree Classification on Heart Disease Dataset

## 🧠 Objective:
Apply **Decision Tree Classification** on the Heart Disease dataset to predict the presence of heart disease, evaluate model performance, and interpret results through visualizations and metrics.

---

## 🛠️ Technologies Used
- Python
- Pandas, NumPy – Data processing
- Scikit-learn – Decision Tree modeling, encoding, evaluation
- Matplotlib, Seaborn – Data visualization
